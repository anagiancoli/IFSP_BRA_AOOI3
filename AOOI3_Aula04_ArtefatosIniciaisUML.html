1.1.1. Artefato: Documento Visão¶

  • É um relato resumido com os principais tópicos em que o negócio a ser automatizado deve fornecer.
  • Normalmente, faz parte de um contrato de desenvolvimento de software.
  • Poderá abordar: aspectos da tecnologia (linguagens, banco de dados, etc) que serão usados.
  • É um documento voltado para os contratantes do software.

  • O diagrama de Caso de Uso (nível 0) ou Pacote de Sistema baseia-se neste documento.

  • Apresenta de uma forma resumida o que o contratante (REQUISITANTE) deseja:

  • É o que o contratado (DESENVOLVEDOR) quer:

  • Nasce na primeira reunião com os clientes de software (pessoas externas, internas, departamentos, etc).

1.1.2. Artefato: Documento Nomenclatura¶

  • Serve para criação de um padrão único de pré e pós-fixação para as palavras envolvidas em estruturas de código.
  • Serve principalmente, para novos colaboradores ao se juntarem a equipe, serem treinados em um mesmo padrão, sem existir vários códigos e estilos.
  • Cada empresa deve adotar uma notação que permita o encontro da identidade do seu formato de nomenclatura.

1.1.3. Artefato: Documento Glossário¶

  • Traduz para as pessoas que não conhecem o software, todos os termos utilizados pelo usuário contratante e outros stakeholders.
  • Cada empresa adota o seu vocabulário.
  • A finalidade deste documento é agir como tradutor entre os participantes da análise, do desenho, da codificação, dos testes e dos herdeiros do software e o vocabulário particular da empresa.
Resumo
Os 3 documentos devem estar disponíveis na nuvem da empresa para consultas de todos os participantes, inclusive os desenvolvedores de software que o tomarão como modelo para desenvolvimento de outros softwares.
- O Documento Visão é o único que exige assinatura do Cliente Contratante, pois ele delimita o escopo do sistema proposto.
- O Documento Nomenclatura deixa claro a forma de escrever o código deve ser única.
- O Documento Glossário evita que entrem sem saber os significados dos termos utilizados no software desenvolvido.
