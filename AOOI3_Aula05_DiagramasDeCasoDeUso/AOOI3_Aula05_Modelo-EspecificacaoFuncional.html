Fluxo Alternativos (Enumerar os passos alternativos segundo o padrão: AXX [PXX] Nome do Fluxo Alternativo)
Exemplo:
A01 [P06] Inserção de um cliente
1. O Responsável pelo setor seleciona a opção de inclusão ou outra opção ou opta por finalizar o caso de uso.
2. O Sistema apresenta a Interface003 para preenchimento dos dados do cliente.
3. O Responsável pelo setor informa os dados: sigla, nome, um endereço, o telefone principal, o e-mail principal, CPF.
4. O Sistema registra as informações e apresenta a MSG001.
5. O caso de uso continua a execução no passo [P04].
A02 [P06] Atualização de um cliente
1. O Responsável pelo setor seleciona a opção de atualização ou outra opção ou opta por finalizar o caso de uso.
2. O Sistema exibe a Interface004 habilitada para o modo editável com as informações pertinentes ao cliente selecionado.
3. O Responsável pelo setor informa as atualizaçãoes a serem feitas.
4. O Sistema registra as alterações.
5. O caso de uso continua a execução no passo [P04].
A03 [P06] Exclusão de um cliente
1. O Responsável pelo setor seleciona a opção de exclusão ou outra opção ou opta por finalizar o caso de uso.
2. O Sistema exibe a MSG002.
3. O Responsável pelo setor confirma a exclusão.
4. O Sistema realiza a exclusão e apresenta a MSG003.
5. O caso de uso continua a execução no passo [P01].
A04 [P06] Consulta de dependentes autorizados
1. O Responsável pelo setor seleciona a opção de consulta de dependentes autorizados ou outra opção ou opta por finalizar o caso de uso.
2. O Sistema aciona o caso de uso estendido UC005 - Consultar Dependentes Autorizados.
3. O caso de uso continua a execução no passo [P04].
